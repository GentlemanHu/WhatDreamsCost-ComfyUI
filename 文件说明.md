# WhatDreamsCost-ComfyUI 节点包

ComfyUI 自定义节点集合，专为 LTX Video (LTXV) 工作流设计。

## 快速开始

将本目录放入 ComfyUI 的 `custom_nodes/` 文件夹后重启 ComfyUI 即可自动加载。

## 项目结构

```

├── ltx_keyframer.py             # LTX Keyframer 节点

├── ltx_auto_sequencer.py        # LTX Auto Sequencer 节点（自动分配帧位置）★ 新增
```

## 节点说明

### LTX Keyframer
将批量图片直接替换到视频 latent 的指定帧位置（噪声遮罩更新）。  
需手动为每张图片指定 `insert_frame_N` 和 `strength_N`。

**输入：** `vae` · `latent` · `multi_input` (IMAGE batch) · `num_images` · 动态 `insert_frame_N` / `strength_N`  
**输出：** `latent`（修改后）

---

### Multi Image Loader
从文件路径列表加载多张图片，支持调整大小、插值方式和 JPEG 压缩。  
提供一个合并的 batch 输出（`multi_output`）以及最多 50 个单独输出（`image_1` … `image_50`）。  
UI 带有拖拽排序图片画廊，支持上传、粘贴、拖入文件。

**输入：** `image_paths` (文本列表) · 尺寸 · 插值 · 调整方式 · 压缩  
**输出：** `multi_output` + `image_1` … `image_50`

---

### LTX Sequencer
将多张图片作为引导关键帧添加到 LTXV 视频生成的 conditioning 中。  
需手动为每张图片指定插入位置（帧数或秒数）和强度。支持 `frames` / `seconds` 两种插入模式。

**输入：** `positive` · `negative` · `vae` · `latent` · `multi_input` · `num_images` · `insert_mode` · `frame_rate` · 动态 `insert_frame_N` / `insert_second_N` / `strength_N`  
**输出：** `positive` · `negative` · `latent`

---

### LTX Auto Sequencer ★ 新增
**自动**将批量图片分配为视频时间线上的引导关键帧，无需手动设置每张图片的帧位置。  
接受来自**任意来源**的批量图片：`MultiImageLoader`、生成节点输出、KSampler 图像等。  
总帧数自动从输入 latent 推算，无需额外配置。

#### 分布模式（`distribution_mode`）

| 模式 | 说明 |
|------|------|
| `even` | N 张图片均匀分布，包含首尾帧（默认） |
| `first_last` | 第 1 张→第 0 帧，最后 1 张→末尾帧，中间图片位置为 0 |
| `first_last_even` | 第 1 张→第 0 帧，最后 1 张→末尾帧，中间图片均匀分布 |
| `fixed_interval` | 从第 0 帧开始，每隔 N 帧/秒放置一张图片 |
| `custom_pattern` | 逗号分隔的自定义位置列表；值比图片数量少时自动循环 |

#### `custom_pattern` 单位（`pattern_unit`）

| 单位 | 格式示例 | 说明 |
|------|---------|------|
| `normalized_0_to_1` | `0, 0.25, 0.5, 1.0` | 0 = 开始，1 = 结束 |
| `absolute_frames` | `0, 24, 48, 96` | 像素空间帧号 |
| `absolute_seconds` | `0, 1.0, 2.5, 4.0` | 时间（秒），需配合 `frame_rate` |

**输入：** `positive` · `negative` · `vae` · `latent` · `images` (任意 IMAGE batch) · `distribution_mode` · `global_strength` · `frame_rate`  
**条件输入（按模式显示）：** `interval` · `interval_unit` · `custom_pattern` · `pattern_unit`  
**输出：** `positive` · `negative` · `latent`

---

### Speech Length Calculator
根据脚本中引号内的台词估算语音时长，输出慢速、平均、快速三档帧数。

**输入：** `text` (含引号台词) · `fps` · `additional_time`  
**输出：** `slow_frame_count` · `average_frame_count` · `fast_frame_count`

## 技术栈

- Python 3.x
- PyTorch
- ComfyUI `comfy_api.latest.io` (V3 节点 API)
- `comfy_extras.nodes_lt.LTXVAddGuide`（LTXV 官方引导帧基类）
- JavaScript ES Module（ComfyUI 前端扩展）

## 注意事项

- **LTX Auto Sequencer** 不影响、不依赖其他已有节点，可独立使用
- `custom_pattern` 值比图片数量少时会**循环**使用（不会报错）
- `fixed_interval` 超出视频末尾的图片会被自动**跳过**并在控制台打印警告
- 所有节点均不修改上游节点的输出（内部对 latent/conditioning 做 clone）
- 本插件由 AI 生成，使用前请在目标环境中测试验证
